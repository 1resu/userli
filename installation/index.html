Installation

Requirements

  • Webserver (e.g Caddy)
  • PHP >= 7.1
  • MariaDB or MySQL
  • libsodium
  • OpenSSL binary (for MailCrypt feature)

Libsodium is already included in PHP 7.2. You can also run this application with PostgreSQL oder SQLite.

    Commands

    This app brings custom commands:

    Checkpassword

    The console command bin/checkpassword is a checkpassword command to be used for authentication (userdb and passdb lookup) by external services. So far, it’s only tested with Dovecot.

    Customize

    You can override translation strings individually by putting them into override localization files at translations/<lang>/messages.<lang>.yml.

    Webserver

    Userli has to be installed like an other Symfony application.
